# Social Network Server
Rest API for a Social Network written with [express](https://github.com/expressjs/express). Includes authentication with [passport](https://github.com/jaredhanson/passport) and session storage in [Redis](https://github.com/redis/redis).

## Requirement
You must have npm and node installed on your system. Also, you must create a .env file and declare the variables shown in the .env.example file

## Usage
```
npm install
npm start
```
